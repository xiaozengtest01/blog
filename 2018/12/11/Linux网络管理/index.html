linux网络管理

  • 2018-12-11

一、安装linux(centos)

1、安装centos6.5或7.0

2、安装ReadHat Enterprise Linux 7.0（RHEL7.0）

安装步骤：
（1）下载CENTOS6.5

（2）在VirtualBox，选择“设置”-“存储”-“光驱”-选择下载好的镜像文件。

（3）修改VirtualBox中的：
设置——系统——处理器——“扩展特性 启用PAX/NX”
启动虚拟机，安装。

（4）安装CENTOS6.5（只能是文本模式）:回车选择第一项安装。

（5）选择“skip”-“ok”-“re-initalize”

（6）修改网络，以便能上网

fconfig eth0 up

vi /etc/sysconfig/network-scripts/ifcfg-eth0
修改

Device=eth0

HWADDR=默认

TYPE=Ethernet

UUID=默认

ONBOOT=yes

NM_CONTROLLED=yes

BOOTPROTO=dhcp

保存：按esc-:wq!回车

#init 6

#ping www.yxnu.edu.cn

在windows登录外网

#ping www.baidu.com

（7）安装GNOME桌面：

yum  groupinstall  “Desktop”

  • 输入：y 确认下载

  • 输入：y 确认安装

  • yum groupinstall “X Window System”

  • yum groupinstall ”Chinese Support”

  • 修改/etc/inittab中的3位5

  • 创建用户：先完成第8步。

  • 重启

二、远程管理Linux系统

1、安装telnet服务（不安全：明文传输）

（1）、查看本机是否安装telnet

#rpm -qa | grep telnet

如果什么都不显示。说明你没有安装telnet

（2）、开始安装

yum install xinetd——–安装internet服务

yum install telnet——–安装telnet客户端（可以不安装）

yum install telnet-server—安装telnet服务器

（3）、装好telnet服务之后，默认是不开启服务的，下面我们需要修改文件来开启服务。

vi /etc/xinetd.d/telnet 修改 disable = yes 为 disable = no

（4）、需要激活xinetd服务

#service xinetd restart 或者 #/etc/rc.d/init.d/xinetd restart

（5）、关闭防火墙：service iptables stop
（6）、在windows上启用telnet：控制面板-程序-程序和功能-打开或关闭windows功能-启用telenet客户端。

注意：把虚拟机的网络改为“桥接”-修改IP地址。
